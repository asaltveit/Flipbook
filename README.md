# Flipbook
A web app that creates flipbooks with AI generated images from your original drawing.  Upload your own drawing and a prompt and the drawings generated will use your style consistently.

Uses Fal, Anthropic, Black Forest (Flux).
Made with Claude.ai, Vercel, Supabase, React, JavaScript, TailwindCSS.

Created during the Hack FLUX: Beyond One hackathon 11/22/25-11/23/25.
